## 1) What have I been doing this week? What skills did I need?

In Week 9, I focused on refining the sections I had already implemented in WordPress, improving visual consistency, and adding more dynamic elements to the page.

A key part of this week was enhancing the interface by introducing motion. I worked on adding subtle animations to make the page feel more interactive and engaging, while still respecting the overall design and user experience.

One of the main technical challenges this week was implementing a moving logo strip. Since there was no pre-built widget available in Elementor for this functionality, I initially attempted to create it without code, relying only on built-in tools.

Although I partially succeeded, I encountered limitations—especially with creating a seamless looping animation. The strip would not loop correctly, which affected the overall experience.

After several attempts and increasing frustration with the limitations of the no-code approach, I decided to implement the component manually using HTML and CSS. This approach proved to be significantly more flexible and efficient.

To achieve this effect, I structured the logos inside a continuously scrolling flex container and duplicated the items to create a seamless loop:

### HTML Structure

~~~html
<div class="logo-strip">
  <div class="logo-track left">
    <div class="logo-item"><img src="https://ehpa.org/wp-content/uploads/2026/04/afpac.org_.png"></div>
    <div class="logo-item"><img src="https://ehpa.org/wp-content/uploads/2026/04/agas.com_.jpg"></div>
    <!-- duplicated logos for seamless looping -->
  </div>
</div>
~~~

### CSS Styling

~~~css
.logo-strip {
  width: 100%;
  overflow: hidden;
  position: relative;
  padding: 6px 0;
}

.logo-track {
  display: flex;
  align-items: center;
  width: max-content;
  gap: 12px;
}

.logo-track.left {
  animation: scrollLeft 35s linear infinite;
}

.logo-item {
  flex: 0 0 auto;
  display: flex;
  align-items: center;
  justify-content: center;
  min-width: 100px;
  height: 44px;
  padding: 6px 10px;
  border-radius: 10px;
  background: rgba(255, 255, 255, 0.6);
}

.logo-item img {
  max-height: 20px;
  max-width: 80px;
  object-fit: contain;
}

@keyframes scrollLeft {
  from { transform: translateX(0); }
  to { transform: translateX(-50%); }
}
~~~

By duplicating the logos and using a continuous translate animation, I was able to create a smooth infinite scrolling effect without visible breaks.

The page is now much more dynamic, although it is not yet fully finalized. Some parts still require refinement:

- The footer still needs further adjustments  
- One section of the page is not yet implemented  
- Buttons and links are not yet interconnected with other pages
